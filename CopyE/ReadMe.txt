CopyE утилита полезная при копировании файлов, когда точное местоположение файла искать затруднительно по разным причинам.(лень)
CopyE ищет файлы указаные в командной строке как source в каталоге и его подкаталогах, указанном как destination
и при нахождении точного совпадения имён файлов копирует файл source в место найденного файла. Заменяются все найденные в destination файлы.
Пример 
CopyE d:\readme.txt c:\     
все файлы readme.txt на диске С будут заменены на файл d:\readme.txt
Допускается использование нескольких source, destination может быть только один, последним в списке аргументов
Пример
CopyE source1 source2 source3  c:\destination
